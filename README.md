# My Django Translator

Django asosida qurilgan oddiy tarjima (translator) ilovasi. Foydalanuvchi matnni kiritib, bir tildan ikkinchisiga tarjima qilish imkoniyatiga ega bo'ladi.

## 📌 Loyiha haqida

Bu loyiha Django framework yordamida backend qismi qurilgan, matnlarni tarjima qilish funksiyasini taqdim etadigan veb-ilova.

## 🛠 Texnologiyalar

- **Python** — asosiy dasturlash tili
- **Django** — backend framework
- **HTML/CSS** — foydalanuvchi interfeysi

## ✨ Xususiyatlari

- Matnni kiritish va tarjima qilish
- Oddiy va tushunarli foydalanuvchi interfeysi
- Django admin panel orqali boshqarish imkoniyati

## 🚀 O'rnatish

```bash
git clone https://github.com/sohibjon1606/my-django-translator.git
cd my-django-translator
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```
